Contexte

SAE 203 - Produire un site web avec une Base de données

Le Brief

Le projet était réaliser un site sur un sujet imposé : les cafés historiques pour moi.

J'ai suivi le cahier des charges en réalisant toutes les pages demandés, en incluant une base de données réalisé sur MySQL. J'ai aussi réalisé la partrie Administration du site qui permet de gérer les bases de données. De plus, j'ai réalisé le dossier d'analyse de mon site, du style graphique choisi aux tests de performance en passant par l'analyse des données

Consulter le dossier d’analyse (PDF)
01

Page d'accueil

Pour l'ensemble du site, je suis partie dans les tons marron et crème pour rappeler le café. Pour la page d'accueil, j'ai choisi de mette un hero avec l'image de l'intérieur d'un café historique pour plonger directement l'utilisateur dans l'ambiance. On retrouve sur ce hero, un bouton qui renvoie directement à la page recherche. Ensuite, j'ai ajouté une section où l'on retrouve 3 villes historiques avec une photo et un petit texte de présentation et un bouton qui permet d'accéder à la page des données. La deuxième section, elle, est sur les personnalités qui ont fréquentés ces cafés avec toujours le bouton qui renvoie à la page des cafés et des personnalités.

02

Page données

Dans cette page de données, les données ont été réalisé en Mysql, avec 2 tables : une sur les cafés historiques et une autre sur les personnalités. La première table comprends : le nom du café, sa date de création, le pays dans lequel il est situé, son style architectural, les personnalités qui l'ont fréquenté, son adresse, ses coordonnées et une photo du café. La seconde table comprends : le nom et le prénom de la personnalité, son métier, le café qu'elle fréquentait, sa première date de visite du café.

03

Page recherche

La page recherche permets eu utilisateurs de rechercher les cafés selon les pays. J'ai intégré une carte intéractive qui est relié aux tables et qui se met à jour lors de l'ajout d'un nouveau café. On y retrouve le nom du café, son pays et sa photo.
